# Лекция 2. Линейные блоковые коды и методы декодирования

## Назначение канальных кодов
В теории кодирования под кодом понимают конечное множество двоичных слов фиксированной длины. Источник информации формирует все возможные информационные слова длины k с одинаковой вероятностью, всего 2^k вариантов.

Перед канальным кодированием избыточность, как правило, устраняется (например, с помощью кодов сжатия). На этапе канального кодирования избыточность, наоборот, вводится специально — для повышения надёжности передачи.

---

## Основные параметры блочного кода
Для любого кода вводятся следующие характеристики:
- **n** — длина кодового слова;
- **k** — количество информационных символов;
- **r = n − k** — число проверочных символов;
- **R = k/n** — скорость кода.

---

## Линейные блоковые коды

### Определение
Код C называется линейным, если поразрядная сумма по mod 2 любых двух кодовых слов также принадлежит C. Иными словами, линейный код представляет собой k-мерное линейное подпространство пространства F₂ⁿ.

### Матричное задание кода

**Порождающая матрица G (k * n)**  
Строки матрицы образуют базис кода. Кодовое слово формируется по правилу:
c = m * G,  
где m — информационное слово длины k.

**Проверочная матрица H ((n−k) * n)**  
Слово является кодовым тогда и только тогда, когда выполняется условие:
c * Hᵀ = 0.

Фундаментальное соотношение:
G * Hᵀ = 0.

Размерность пространства проверок равна n − k, что соответствует числу проверочных символов.

---

## Систематическое представление кода

На практике наиболее удобен систематический вид матриц.

**Порождающая матрица:**
G = [Iₖ | P],  
где Iₖ — единичная матрица.

**Проверочная матрица:**
H = [Pᵀ | Iᵣ].

В этом случае первые k символов кодового слова совпадают с исходной информацией, а оставшиеся r являются проверочными.

Коды, получаемые друг из друга перестановкой столбцов и элементарными преобразованиями строк, считаются эквивалентными и имеют одинаковые параметры (n, k, dₘᵢₙ).

---

## Минимальное расстояние и его свойства

Для линейного кода минимальное расстояние равно минимальному ненулевому весу кодовых слов:
dₘᵢₙ = min ω(c), c ≠ 0.

Удобный способ вычисления через проверочную матрицу:
минимальное расстояние равно наименьшему числу линейно зависимых столбцов матрицы H.

**Критерий:**
- любые d−1 столбцов H линейно независимы;
- существует набор из d линейно зависимых столбцов.

**Граница Синглтона:**
dₘᵢₙ ≤ n − k + 1.

---

## Методы декодирования линейных кодов

### Табличное декодирование
Пространство F₂ⁿ разбивается на смежные классы по коду. Для каждого класса выбирается лидер — вектор минимального веса.

Процедура декодирования:
1. Определяется смежный класс принятого вектора.
2. В качестве ошибки выбирается лидер этого класса.
3. Переданное слово получается как r ⊕ e.

### Синдромное декодирование
Синдром принятого вектора r вычисляется как:
S = r * Hᵀ.

Основные свойства:
- синдром зависит только от ошибки;
- все векторы одного смежного класса имеют одинаковый синдром;
- каждому синдрому соответствует ровно один лидер смежного класса.

Данный метод является основным на практике.

---

## Простейшие линейные коды

### Код с проверкой на чётность
Код параметров (n, n−1) с проверочной матрицей, состоящей из одной строки единиц.

Свойства:
- минимальное расстояние равно 2;
- не исправляет ошибки;
- обнаруживает любые ошибки нечётного веса.

---

## Коды Хэмминга

### Построение
Проверочная матрица H формируется из всех ненулевых двоичных векторов длины r, взятых в качестве столбцов.

### Параметры
- n = 2ʳ − 1;
- k = 2ʳ − 1 − r;
- dₘᵢₙ = 3.

Коды Хэмминга исправляют любые одиночные ошибки.

### Декодирование
Синдром совпадает со столбцом матрицы H, соответствующим позиции ошибки, что позволяет сразу её исправить.

Коды Хэмминга являются оптимальными среди кодов с теми же параметрами длины и расстояния.

---

## Модификации кодов Хэмминга

**Симплексный код**  
Дуальный по отношению к коду Хэмминга, имеет параметры (2ʳ−1, r). Все ненулевые кодовые слова обладают одинаковым весом.

**Расширенный код Хэмминга**  
Получается добавлением дополнительного проверочного символа. Имеет параметры (2ʳ, 2ʳ−r−1) и минимальное расстояние 4.

**Дуальный код к расширенному коду Хэмминга**  
Известен как код Рида–Маллера первого порядка RM(1, r).
